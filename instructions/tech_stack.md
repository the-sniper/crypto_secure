# Project Context

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn UI
- **Icons:** Lucide React
- **Package Manager:** npm

## Conventions
- Use functional components.
- Use `lucide-react` for icons.
- Prefer server components where possible.
- Use `kebab-case` for file names.
